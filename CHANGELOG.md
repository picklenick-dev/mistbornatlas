# Changelog

## Milestone 3: Open-Source

Centralised character configuration into `characterConfig.ts`, migrated all icons to Font Awesome 6, and consolidated SCSS architecture with shared animations and mixins. Removed dead code, deduplicated constants and helpers, extracted shared components, and migrated inline styles to SCSS modules. Set up Prettier, EditorConfig, and ESLint. Deployed via GitHub Actions to Azure Static Web Apps with lazy-loaded locale chunks and `React.lazy()` components, reducing the initial JS bundle from 1,268 kB to ~460 kB (-64%). Added Apache 2.0 license with full README documentation. **Sub-chapter movement** system allowing characters to appear at multiple locations within a single chapter, each with linked popups and "Part X/Y" indicators.

---

## Milestone 2: Polish, Localization & Refinement

The community came through - Mistborn Atlas now lives at **[www.mistbornatlas.com](https://www.mistbornatlas.com)**, bought with Ko-fi support. The entire UI and all in-world content now has translations for **11 languages** (English, Spanish, French, German, Polish, Portuguese, Russian, Chinese, Japanese, Arabic with full RTL support, and Swedish) - AI-generated first pass based on official publisher editions, not yet verified by native speakers. Full i18n system extracting every hardcoded English string through `useLanguage()`, with complete translation files using publisher-specific terminology and confidence tracking. Translated all 173 character movement entries, 44 world locations, 3 cities, and ~67 city landmarks per locale. Audited translations against official publisher editions. Added localized SEO metadata with hreflang links and a language picker with native language names. Added a dedicated **Secret History toggle** hidden behind spoiler mode so readers who haven't read Secret History won't see anything they shouldn't. Entering and exiting city maps now has smooth zoom transitions with a misty blur effect - the destination materialises through the mist. Characters earn **title progressions** as the story advances, tied to specific chapters so titles are never shown before they're earned. Dual portrait system with Vin transitioning from street urchin to Mistborn and Marsh's appearance changing as the story does. Larger pulsating character markers (44px), subtler hover dimming, Luthadel-focused first-visit experience, automatic timeline pause on first city prompt, and lots of mobile UX improvements. Corrected movement data including gate names, character positions, and landmark placements verified against source material.

## Milestone 1: Core Map & Timeline

The core **character tracking system** is working - follow a character as the timeline plays and the map pans to them at each chapter, with pause-on-read so you can read each event before advancing. Character popups have "Earlier / Later" buttons to browse past movements, and when multiple characters share a location their markers spread out. Hovering a character dims everything else to highlight their journey. All location markers now use **Allomantic metal symbols** from Isaac Stewart's Steel Alphabet artwork - Dominance Seats get Gold, Ashmounts use Atium, Fortresses use Steel, each with their own metallic disc. An atmospheric layer of **falling ash and creeping mists** drifts across the map (interactive too - move your cursor and particles react like you're Steelpushing them). **68+ new locations** added across the world map (Valtroux, Haverfrex, Arguois Caverns, Kandra Homeland, Terris Mountains, and more), 20 Luthadel landmarks (Fountain Square, Canton of Inquisition, Sootwarrens), 14 Urteau landmarks (Spook's Hideout, the Burning District), and 14 Fadrex City landmarks (Storage Cavern, Yomen's Palace). Spoiler mode hides character portraits and names before their debut chapter with a "?" placeholder. Six characters tracked with Coppermind portrait markers: Vin, Kelsier, Sazed, Elend, Marsh, and Spook.
